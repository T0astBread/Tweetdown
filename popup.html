Tweetdown / Usage

Tweetdown lets you use basic Markdown features on Twitter.

Markdown is a neat little markup language to add style and meaning to your text. To use these features, surround a word (or any string of characters) with Markdown tags.

Supported Markdown tags:

Preview Raw Description
Bold **Bold** or __Bold__ Renders text in bold characters (often used to highlight it)
Italic *Italic* or _Italic_ Renders text in italic characters
Italic and Bold ***Italic and Bold*** or ___Italic and Bold___ Renders text in italic and bold characters
// Code Code (inline) `Code (inline)`

Represents text in a way in which computer code is often represented.

Note that this is the inline version of the code tag, so it should only be used on single line code snippets (although it will be recognized as a code block). If you want (proper) multiline code use a code block.
